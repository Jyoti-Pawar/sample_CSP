Artificial Intelligence/ Machine Learning

John McCarthy, usually known as one of the godfathers of Artificial Intelligence (AI), defined it as “the science and engineering of making intelligent machines.” Information is a vital part of AI research. Machines and programs need to have abundant data associated to the world to act and react like human beings. AI will have access to groups, assets, things and relations between all of them to implement knowledge engineering. AI initiates joint sense, problem-solving and analytical reasoning power in machines. Machine Learning (ML) is a science of designing and applying algorithms that can learn things from past cases.

    Subjects

    Introduction to AI/ML

    This subject gives an overview on how to create human interfaces applications which are simple to program, easily understandable and maintainable. It provides an overview of AI concepts and workflows, machine learning, deep learning and performance metrics.

    Machine Learning

    This subject provides an understanding of machine learning algorithms. Here the audience will learn to explore machine learning techniques (supervised and unsupervised learning) and statistical concepts (probability, inference, modelling) and apply them in practice.

    Industry Application Development using Machine Learning

    This subject provides an overview on using the machine learning technology to build an application for various industries like Healthcare, Manufacturing, Security, Transportation, Agriculture, Retail, Oil and Gas and others.
