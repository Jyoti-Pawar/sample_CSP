Artificial Intelligence

John McCarthy, usually known as one of the godfathers of AI, defined it as “the science and engineering of making intelligent machines.” Information is a vital part of AI research. Machines and programs need to have abundant data associated to the world to act and react like human beings. AI will have access to groups, assets, things and relations between all of them to implement knowledge engineering. AI initiates joint sense, problem-solving and analytical reasoning power in machines. ML is a science of designing and applying algorithms that can learn things from past cases.

This specialization helps to gain knowledge on how artificial intelligence makes it possible for machines to learn from experience, adjust to new inputs and perform human-like tasks. The audience will learn about the technologies like machine learning, deep learning and natural language processing. It also provides an overview of how these technologies are used for training the systems to accomplish the specific tasks by processing large amounts of data and recognizing patterns in the data.

Subjects

Introduction to AI/ML

This subject gives an overview on how to create human interfaces applications which are simple to program, easily understandable and maintainable. The specialization provides an overview of AI concepts and workflows, machine learning, deep learning, and performance metrics.

Machine learning

This subject provides an understanding of machine learning algorithms. Here the audience will learn to explore machine learning teachniques (supervised and unsupervised learning) and statistical concepts (probability, inference, modelling) and apply them in practice.

Industry application development using machine learning

This subject provides an overview on using the machine learning technology to build an application for various industries like Healthcare, Manufacturing, Security, Transportion, Agriculture, Retail, Oil and Gas and others.
